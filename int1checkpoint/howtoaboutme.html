How I Created a Navbar

To create a navbar, I made another file called index.css and linked it to my current file index.html. I used the <a href> tag, <li> tag, and <ul> to create my navbar. When I created my navbar I put the <ul> first then I went to the next line and put in the <a href => and put in the name of my file. After I put my <li> and put what I wanted to named my first part of the navbar. On my eighteenth line I put <a href> again and put the name of my file. Lastly I put another <li> and put the name of my second navbar and closed it with </a> and then </ul>.

How I Created a Header

To create a header you put <h>. You can change the size of your header by putting 1,2,3 etc. After you chose your size for your header, put your information in.

How I Created a Paragraph

To create a paragraph you have to use a <p> tag and then you can put all your information in you paragraph.

How I Used CSS

How I Put a Background Color and Border Around My Text

Put a div around all of your paragraph ad use an ID tag to name the div so you can add a background color.

After go to your CSS file and write # and then the name of the your ID tag. Put an Ellipsis right after that and press enter. In the next line write background: (insert color name);. Go to the next line and put border: solid (color name) 5px;.

How I Centered Things

Give you information an ID tag and then go to you CSS file. In the CSS file insert #(name of ID tag) then put text-align: center;

How I Moved the Margin

To move the margin add an ID tag to your paragraph, header, image or div. Then go to your CSS file and put #(name of your ID tag) { press enter and then write margin-left: or margin-right:. After margin-left/right: put the number of pixels you want the information to move by writing (number) px.
