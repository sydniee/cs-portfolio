What I Made

I created a watermelon that appears at the spot where you click. After a certain amount of clicks all the watermelons on the screen will disappear. Everytime you click for a watermelon to appear, the seeds will always show up in a different spot.

How I Made the Base of My Watermelon

To make the base of my watermelon, I had to make a circle. To make a circle you use an ellipse. An ellipse contains 4 things x,y, width, and height. When you write out your code you write ellipse(x,y,width,height). The x and y determines where on the screen the circle will appear and the width and height determines the size of the circle. When I made the green base, I wanted to make it a certain size so I changed my width and height which looked like ellipse(x,y,150,1500). I had to color my base so I wrote fill(70,242,147) which colored the inside of my watermelon green. After, I had to add the inside of the watermelon so I made another ellipse with a smaller height and width which was ellipse(x,y,125,125). Next I had to color it so I used another fill but changed the numbers to fill(226,63,88). After that I wanted to change the color of the stroke so I wrote stroke and wrote my chosen color which was white, stroke(255). To make it thicker I just changed the strokeweight by writing strokeWeight (3).

How to Make the Seeds Appear Randomly With a Loop

To make the seeds appear randomly I had to make a loop. The loop was to make my seeds always appear in a random and different spot whenever I clicked for a watermelon to appear. The loop I used was a for loop. In my code I wrote for(var i= 0; i < random(8-20); i +=1 ). That meant if my variable i was less than then a random 8-20 seeds would appear and once the 8-20 seeds appear it repeats the process mkaking a loop.

How to Clear the Background

To clear the background you use an if else statement. I made a variable called numWatermelons which equaled to 0. Then for my code I wrote if (numWatermelons > 14) { background(); numWatermelons= 0;} This means if the number of watermelons on the screen is more than 14 then the background turns back to green and the number of watermelons reset to 0.
